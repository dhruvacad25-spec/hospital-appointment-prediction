# Hospital Appointment No-Show Prediction
-----
This project uses Machine Learning techniques to predict whether a patient is likely to miss a scheduled hospital appointment. By analyzing factors such as age, gender, appointment scheduling details, medical conditions, and SMS reminders, the model identifies patterns associated with missed appointments.

## Objectives
- Analyze patient appointment data
- Perform data cleaning and preprocessing
- Visualize important trends and patterns

## Model Performance and Limitations

The project uses a K-Nearest Neighbors (KNN) classifier to predict whether a patient is likely to attend or miss a hospital appointment.

During experimentation, KNN achieved the highest overall accuracy among the tested algorithms:

- KNN: 82.65%
- Decision Tree: 70.55%

However, the dataset used for training was highly imbalanced, with the majority of appointments being successfully completed. As a result, the model tends to favor the majority class during prediction.

This project was developed as part of a Data Analysis and Machine Learning Bootcamp to demonstrate the complete end-to-end workflow, including:

- Data preprocessing and cleaning

- Model training and evaluation
